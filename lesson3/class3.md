# class4

## html

1. HypterText:从一个链接到另一个链接（a 标签）
2. Markup :用不同的 Tags 实现标记
3. Language：一门语言
4. 核心特性：Semantics（语义化）
   A11Y：可访问性（accessibility）帮助残障人士拥有上 web 服务器的能力
5. <!DOTTYPE html>:声明自动识别html5的规则
6. 服务端约定默认打开 index.html 文件
7. 重要标签
   header：头部标签
   nav:索引
   main:主体（最好只写一个）
   article：文章
   section:章节
   aside:在旁边
   footer:底部信息
