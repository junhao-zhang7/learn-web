1. 光猫：接受运营商光信号，转化成网线传输的电信号
2. 路由器：连接多个网络并转发数据
3. SSID：Wi-Fi名称
4. wifi noise：周围其他路由器或电子设备（如微波炉，蓝牙）产生的无线信号干扰，会降低WiFi网速和干扰，越靠近-100越好
5. RSSI 是 Received Signal Strength Indicator 的缩写，中文叫接收信号强度指示，是衡量设备接收到的无线信号（如Wi-Fi、蓝牙）强弱的关键指标。

一、核心要点（必懂）

1. 本质：表示设备与路由器之间无线信号的强弱程度，数值通常为负数。
2. 数值解读（Wi-Fi场景）：

- -30 dBm：信号极强（理论最大值，几乎只有紧贴路由器时能达到）。
- -50 ~ -60 dBm：信号很好，网速稳定，无卡顿。
- -60 ~ -70 dBm：信号良好，日常使用完全足够。
- -70 ~ -80 dBm：信号一般，可能出现偶尔卡顿，网速下降。
- -80 ~ -90 dBm：信号较差，频繁卡顿，甚至掉线。
- -100 dBm 及以下：信号极弱，基本无法连接或使用。
  ✅ 结论：数值越接近0（如-50），信号越强；数值越小（如-90），信号越弱。
  3. 单位：dBm（分贝毫瓦），是一个相对值，用于表示功率的大小。

6.  20MHz 是Wi-Fi的信道带宽，指无线信号传输时占用的频率宽度，是决定Wi-Fi速度和穿墙能力的核心参数之一，和你之前问的Channel（信道） 直接相关。

一、核心解析（Wi-Fi场景，必懂）

1. 本质与作用

- 带宽：类比为公路的车道宽度，带宽越大，同一时间能传输的数据越多，网速就越快。
- 单位：MHz（兆赫兹），是频率的单位，代表频率的范围大小。

2. 主流带宽规格（2.4G/5G均支持）

带宽规格 20MHz 40MHz 80MHz 160MHz
传输速度 慢（基础） 快（2倍左右） 更快（4倍左右） 最快（8倍左右）
穿墙能力 强（信号衰减小） 中 弱 极弱
干扰程度 小（占用频率窄） 中 大 极大
适用频段 2.4G/5G 2.4G/5G 主要5G 仅5G

3. 20MHz的核心特点（重点）

✅ 优势：

- 穿墙能力最强：信号在穿墙、远距离传输时衰减最少，适合大户型、远距离使用。
- 抗干扰性最好：占用频率窄，不易和周围路由器的信号重叠干扰，适合路由器密集的环境（如公寓楼）。

❌ 劣势：

- 速度最慢：是基础带宽，最大传输速率远低于40/80MHz，适合对网速要求不高的场景（如手机日常上网）。

二、和Channel（信道）的关键关联

1. 20MHz与信道的关系：

- 20MHz 是标准信道宽度，一个20MHz带宽对应一个独立信道（如2.4G的1、6、11信道）。
- 40MHz 是将两个相邻的20MHz信道合并（如2.4G的1+5信道），提升速度，但会占用更多频率，干扰也更大。
  2. 2.4G频段的特殊限制：
- 2.4G频段带宽窄，仅支持20MHz和40MHz，且40MHz容易和其他信道重叠，干扰严重。
- 因此2.4G频段优先选择20MHz，稳定性远大于速度；5G频段则建议选40/80MHz，追求高速。

三、实用选择建议（结合场景）

1. 选20MHz的情况：

- 设备离路由器距离远、穿墙多（如卧室离客厅路由器隔2堵墙）。
- 周围路由器多、干扰大（如公寓楼、写字楼）。
- 使用2.4G频段（智能家居、老旧设备）。
  2. 选40/80MHz的情况：
- 设备离路由器近、无遮挡（如同一房间）。
- 使用5G频段（手机、电脑高速上网、看4K视频）。
- 对网速要求高（如下载、游戏）。

简单类比

20MHz就像单车道公路，虽然车流量小（速度慢），但道路窄，不容易堵车（抗干扰），且能通到更远的地方（穿墙好）；40/80MHz则像多车道高速，车流量大（速度快），但道路宽，容易堵车（干扰大），且只能在近距离通行（穿墙差）

7. mac地址:分配给网络接口控制器的物理地址
   以太网->局域网
   IP地址->外部网络
8. MAC地址 是 Media Access Control Address 的缩写，中文叫媒体访问控制地址，是设备的物理地址，相当于网络设备的身份证号，全球唯一。

一、核心要点（必懂）

1. 本质与特性

- 全球唯一：由设备厂商在生产时烧录，无法修改（软件伪装除外），确保世界上没有两台设备的MAC地址完全相同。
- 固定性：属于硬件地址，和设备绑定，不会随网络环境变化（如换Wi-Fi、换路由器，MAC地址不变）。
- 格式：由 12位十六进制数 组成，通常分为6段，用冒号（:） 或横杠（-） 分隔，例如： 00:1A:2B:3C:4D:5E 。
- 前6位：厂商标识（OUI），代表生产厂商；后6位：设备序列号，唯一标识该设备。

2. 核心作用

1. 局域网内设备识别：在同一网络（如家里的Wi-Fi）中，路由器通过MAC地址区分不同设备（如你的手机、电脑、电视），是设备间通信的基础。
2. MAC地址过滤：路由器的重要功能，可通过允许/禁止特定MAC地址接入网络，实现网络访问控制（比如禁止陌生设备连Wi-Fi）。
3. 绑定IP地址：将MAC地址与IP地址绑定（静态IP），确保设备每次联网都分配到固定IP，提升网络稳定性。

二、和之前概念的关联（Wi-Fi/路由器场景）

1. 与IP地址的区别（最易混淆，重点）：
特性 MAC地址（物理地址） IP地址（逻辑地址）
本质 硬件标识，全球唯一 网络逻辑标识，可动态分配
修改性 硬件固定，无法原生修改 可手动/自动修改（如DHCP分配）
作用范围 局域网内识别设备 互联网/局域网中定位设备
类比 设备的身份证号 设备的网络门牌号
2. 与Wi-Fi连接的关系：
设备连接Wi-Fi时，首先会向路由器发送MAC地址，路由器验证后（如是否在过滤列表中），才会分配IP地址，完成连接。

三、实用场景（日常能用）

1. 查看设备MAC地址：排查网络问题、设置路由器过滤时常用。
2. 路由器MAC过滤：在路由器后台添加允许的MAC地址，防止别人蹭网，提升网络安全性。
3. 静态IP绑定：将设备MAC与IP绑定，避免IP冲突，适合智能家居、监控设备。

四、如何查看MAC地址？

1. Windows电脑

- 方法1（命令行）：打开CMD，输入  ipconfig /all ，找到物理地址即为MAC地址。
- 方法2（图形界面）：设置 → 网络和Internet → WLAN → 硬件属性，查看物理地址。

2. 手机（安卓/iOS）

- 安卓：设置 → WLAN → 已连接的Wi-Fi详情 → 查看MAC地址。
- iOS：设置 → 通用 → 关于本机 → 找到Wi-Fi地址（即MAC地址）。

3. 路由器

路由器自身也有MAC地址，可在路由器底部标签或后台管理页面查看。

简单类比

MAC地址就像每个人的身份证号，终身不变，用于身份识别；IP地址就像家庭住址，可以搬家（更换），用于定位位置。路由器通过“身份证号（MAC）”认识设备，再通过“住址（IP）”给设备发送数据。9. 手机接入路由器会获得IP，子网页码，网关，DNS
